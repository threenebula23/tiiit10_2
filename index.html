Документация для проекта tiiit10_2

Этот проект реализует алгоритм пузырьковой сортировки на C++ для сортировки массива целых чисел по возрастанию.

Обзор

Файл main.cpp содержит программу, которая:

  • Определяет массив целых чисел.
  • Выводит исходный массив.
  • Сортирует массив с помощью алгоритма пузырьковой сортировки.
  • Выводит отсортированный массив.

Использование

Для компиляции и запуска программы выполните:

g++ main.cpp -o main
./main

Пример вывода:

Исходный: 64 34 34 25 61 12 22 11 90 42 99
Результат: 11 12 22 25 34 34 42 61 64 90 99

Описание алгоритма

Алгоритм пузырьковой сортировки сравнивает соседние элементы массива и меняет их местами, если они находятся в неправильном порядке. Процесс повторяется, пока не потребуется больше перестановок.
